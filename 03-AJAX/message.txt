<h1>AJAX</h1>
<p>AJAX stands for Asynchronous JavaScript And XML</p>
<p>It is the use of the XMLHttpRequest object to communicate with a server</p>
<p>AJAX is not a programming language</p>
<p>AJAX can be sued to send and receive information in various formats, such as XML or JSON or HTML or text files</p>
<p>AJAX is a term to define how browsers can access web servers from a web page</p>
